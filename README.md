# rejectaid- (ON GOING)
Built a Flask-based web app to check eligibility and explain rejection reasons for government schemes.
Why Rejected? Assistant

A simple Flask-based web application that helps users understand why their application for a scheme was rejected based on eligibility criteria.

Features
Input age, income, and category
Checks eligibility against predefined rules
Displays clear rejection reasons or approval status
Tech Stack
Python (Flask)
HTML
JSON
Project Structure
project-folder/
│
├── app.py
├── schemes.json
└── templates/
    ├── index.html
    └── result.html
How to Run
Install Flask:
pip install flask
Run the app:
python app.py
Open in browser:
http://127.0.0.1:5000
Purpose

This project aims to improve transparency in application processes by clearly explaining rejection reasons.
